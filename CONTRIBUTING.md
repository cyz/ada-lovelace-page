# Como contribuir?

Este é um projeto totalmente livre que aceita contribuições via pull requests no GitHub. Este documento tem a responsabilidade de alinhar as contribuições de acordo com os padrões estabelecidos no mesmo. Em caso de dúvidas, [abra uma issue](https://github.com/cyz/ada-lovelace-page/issues/new).

## Passos

1. Crie um fork deste repositório.
2. Envie seus commits em português.
3. Insira um pequeno resumo sobre o que foi adicionado.
4. Solicite um pull request.
